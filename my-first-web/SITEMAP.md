# 🗺️ 개인 블로그 - 페이지 맵 & 유저 플로우

## 📋 전체 페이지 맵

| 페이지 이름 | URL | 설명 | 상태 |
|-----------|-----|------|------|
| **홈** | `/` | 블로그 소개, 최신 글 미리보기 | ✅ 완성 |
| **글 목록** | `/posts` | 모든 게시글을 카드 형태로 표시 | ✅ 완성 |
| **글 상세** | `/posts/[id]` | 선택한 글의 전체 내용 표시 | ✅ 완성 |
| **글 작성** | `/posts/new` | 새로운 글 작성 폼 | ✅ 완성 |
| **글 수정** | `/posts/[id]/edit` | 기존 글 수정 폼 | ✅ 완성 |
| **로그인** | `/auth/login` | 로그인 페이지 | ✅ 완성 |
| **회원가입** | `/auth/signup` | 회원가입 페이지 | ✅ 완성 |
| **마이페이지** | `/mypage` | 내 글 목록, 프로필 | ❌ 미구현 |
| **마이페이지 - 글 관리** | `/mypage/posts` | 내가 작성한 글만 표시 | ❌ 미구현 |

---

## 👥 주요 유저 플로우 (3가지 시나리오)

### 1️⃣ 글 읽기 플로우 (로그인 안 함)

```
홈 (/) 
  ↓
글 목록 (/posts) → 글 카드 선택
  ↓
글 상세 (/posts/[id]) → 내용 읽기
  ↓
[목록으로 돌아가기] → 글 목록으로 이동
```

**특징:** 누구나 접근 가능, 읽기만 가능

---

### 2️⃣ 글 작성 플로우 (로그인 후)

```
로그인 (/auth/login)
  ↓
홈 (/)
  ↓
[새 글 쓰기] 클릭
  ↓
글 작성 페이지 (/posts/new) → 제목/내용 입력
  ↓
[저장] 버튼
  ↓
글 상세 (/posts/[id]) → 작성한 글 확인
```

**특징:** 로그인 필수, 글 작성 권한 필요

---

### 3️⃣ 마이페이지 플로우 (내 글 관리)

```
로그인 (/auth/login)
  ↓
네비게이션 [마이페이지] 클릭
  ↓
마이페이지 (/mypage) → 프로필, 글 개수
  ↓
[내 글 관리] 클릭
  ↓
마이페이지-글관리 (/mypage/posts) → 내 글 목록
  ↓
글 선택 → /posts/[id]
  ↓
[수정] 또는 [삭제]
```

**특징:** 로그인 필수, 자신의 글만 관리 가능

---

## 🚀 네비게이션 구조

```
[로고/홈] | [블로그] | [새 글 쓰기] | [로그인] / [마이페이지]
```

**로그인 전:** `[홈] [블로그] [새 글 쓰기] [로그인]`

**로그인 후:** `[홈] [블로그] [새 글 쓰기] [마이페이지] [로그아웃]`

---

## 💡 추천: 빠진 페이지 & 기능

### 1. **마이페이지 (`/mypage`)** ⭐ 중요
- **이유:** 사용자가 자신의 정보와 글을 관리할 수 있어야 함
- **구성:**
  - 프로필 섹션 (이름, 이메일)
  - 작성한 글 개수, 댓글 개수 (통계)
  - [내 글 관리] 버튼
  - [프로필 수정] 버튼

### 2. **마이페이지-글관리 (`/mypage/posts`)** ⭐ 중요
- **이유:** 내가 작성한 글만 한 곳에서 관리
- **구성:**
  - 내 글 목록 (테이블 또는 카드)
  - 각 글의 [수정], [삭제] 버튼
  - 빠른 수정/삭제 기능

### 3. **404 에러 페이지 (`/not-found.tsx`)**
- **이유:** 존재하지 않는 페이지 접근 시 사용자 경험 개선
- **구성:** 홈으로 돌아가기 버튼

### 4. **검색 기능 (`/search`)**
- **이유:** 글이 많아질수록 필요 (선택사항)
- **구성:** 키워드로 글 검색

### 5. **로그아웃 기능**
- **이유:** 로그인 후 로그아웃 필요
- **위치:** 네비게이션 바의 [로그아웃] 버튼
